# Генератор и проверка надёжности паролей

Простое PHP-приложение для генерации паролей с заданными параметрами и оценки их надёжности.

## Возможности

### Генерация пароля
- Генерация пароля заданной длины
- Настройка используемых символов:
  - Строчные латинские буквы (всегда включены)
  - ПРОПИСНЫЕ латинские буквы (опционально)
  - Цифры (опционально)
  - Специальные символы `!@#$%^&*` (опционально)
- Детерминированная генерация на основе сида (seed):
  - Одинаковый сид всегда даёт одинаковый пароль
  - Автоматическая обработка крайних случаев (отрицательные значения, ноль)

### Проверка надёжности пароля
Оценка пароля по 5 критериям (1 балл за каждый):
- Длина не менее 8 символов
- Наличие строчной латинской буквы
- Наличие ПРОПИСНОЙ латинской буквы
- Наличие цифры
- Наличие специального символа

**Вердикты:**
- 0–2 балла: Слабый
- 3 балла: Средний
- 4 балла: Надёжный
- 5 баллов: Очень надёжный

## Использование

### `generatePassword(int $length, int $seed, bool $useUppercase = true, bool $useDigits = true, bool $useSpecial = false): string`

**Параметры:**

| Параметр       | Тип    | По умолчанию | Описание                               |
|----------------|--------|-------------|----------------------------------------|
| `$length`      | `int`  | —           | Длина пароля                           |
| `$seed`        | `int`  | —           | Сид для детерминированной генерации    |
| `$useUppercase`| `bool` | `true`      | Включить ПРОПИСНЫЕ латинские буквы     |
| `$useDigits`   | `bool` | `true`      | Включить цифры                         |
| `$useSpecial`  | `bool` | `false`     | Включить спецсимволы `!@#$%^&*`        |

```php
// Базовое использование (только строчные буквы)
echo generatePassword(10, 12345);

// С ПРОПИСНЫМИ буквами
echo generatePassword(10, 12345, true, false, false);

// С цифрами и ПРОПИСНЫМИ буквами
echo generatePassword(10, 12345, true, true, false);

// Со всеми типами символов
echo generatePassword(10, 12345, true, true, true);

// Обработка крайних случаев
echo generatePassword(-5, 42);  // Вернёт пустую строку
echo generatePassword(8, 0);     // Сид 0 преобразуется в 1
echo generatePassword(8, -42);   // Сид -42 преобразуется в 42
```

### `checkPassword(string $password): string`

**Параметры:**

| Параметр     | Тип      | По умолчанию | Описание               |
|-------------|----------|-------------|------------------------|
| `$password` | `string` | —           | Пароль для проверки    |

Возвращает строку в формате `"<вердикт> пароль (оценка <N> из 5)"`.

```php
echo checkPassword('abc');           // "Слабый пароль (оценка 1 из 5)"
echo checkPassword('abcdefgh');      // "Слабый пароль (оценка 2 из 5)"
echo checkPassword('abcdef1234');    // "Средний пароль (оценка 3 из 5)"
echo checkPassword('Abcdef1234');    // "Надёжный пароль (оценка 4 из 5)"
echo checkPassword('Abcdef123!');    // "Очень надёжный пароль (оценка 5 из 5)"
```

### Пример полного сценария
```php
// Генерируем надёжный пароль
$password = generatePassword(12, time(), true, true, true);
echo "Сгенерированный пароль: $password\n";

// Проверяем его надёжность
echo checkPassword($password);
```

## Особенности реализации
- Детерминированность: Генерация на основе линейного конгруэнтного генератора (LCG) с фиксированным сидом
- Безопасность: Проверка паролей без хранения в открытом виде
- Обработка ошибок: Корректная работа с невалидными входными данными
